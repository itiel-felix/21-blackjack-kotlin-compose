# Blackjack 21 Android App

This project is an Android implementation of the classic **Blackjack (21)** card game.

The application was developed using **Jetpack Compose** and is compatible with **Android 7.0 and later**.  
Game history is stored locally using **SQLite**.

## Technologies Used

- Kotlin
- Jetpack Compose
- SQLite
- Android Studio

## Features

- Classic Blackjack gameplay
- Draw card or stand mechanics
- Winner detection
- Game history storage
- Simple and clean UI
